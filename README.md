# Dittertp Php

## How do I install these formulae?

`brew install dittertp/php/<formula>`

Or `brew tap dittertp/php` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "dittertp/php"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
